# Spotify-Content-Library-Analysis-Visualization
# Spotify Dashboard | Power BI

## 📌 Project Overview

The **Spotify Top 50 World Dashboard** is a well-designed, fully interactive Power BI dashboard that provides deep insights into global music trends using Spotify's Top 50 chart data.

Built with a modern design philosophy, this dashboard combines **highly realistic visuals**, intuitive navigation, dynamic filtering, and meaningful KPIs to help users explore chart performance, artist popularity, song characteristics, and release trends with ease.

Whether you're a music enthusiast, data analyst, student, or recruiter, this project demonstrates how raw streaming data can be transformed into actionable business intelligence through Power BI.

---

# 📊 Dashboard Features

### 🎯 Interactive Analytics

* Fully interactive report pages
* Cross-filtering across all visuals
* Dynamic slicers for detailed exploration
* Drill-down capabilities
* Responsive visual interactions

### 📈 Key Performance Indicators (KPIs)

* Total Songs
* Unique Songs
* Total Artists
* Average Popularity
* Average Chart Position
* Number of #1 Songs
* Average Song Duration
* Explicit Song Percentage
* Average Album Tracks
* Chart Coverage Duration

---

# 📂 Dataset Information

The dashboard is built using the **Top-50-world** dataset.

### Dataset Columns

| Column          | Description                                          |
| --------------- | ---------------------------------------------------- |
| date            | Date when the song appeared on the chart             |
| position        | Daily chart ranking (1–50)                           |
| song            | Song title                                           |
| artist          | Artist name                                          |
| popularity      | Spotify popularity score                             |
| duration_ms     | Song duration in milliseconds                        |
| album_type      | Album, Single, or Compilation                        |
| total_tracks    | Number of tracks in the album                        |
| release_date    | Official release date                                |
| is_explicit     | Indicates whether the song contains explicit content |
| album_cover_url | Album artwork URL                                    |
